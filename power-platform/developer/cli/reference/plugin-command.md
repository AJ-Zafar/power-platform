# Plugin

Command to create a [plug-in](/power-apps/developer/data-platform/plug-ins) project.

## Parameters

|Parameter Name|Description|Example|
|-------------|-----------|--------|
|init|Initializes a directory with a new plug-in class library.|`pac plugin init`|
